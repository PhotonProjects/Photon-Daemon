# Photon Daemon

Photon Daemon is the server control plane for Photon Panel, built for the
rapidly changing gaming industry and designed to be highly performant and
secure. It provides an HTTP API allowing you to interface directly with
running server instances.

In addition, the daemon ships with a built-in SFTP server.
